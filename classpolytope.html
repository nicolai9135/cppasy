Detailed Description

The purpose of this tool is to perform parameter synthesis in a user-provided region. To do so, the initial region is splitted multiple times into smaller regions. polytope is the class to represent one of these regions, resp.

virtual z3::expr_vector polytope::get_boundaries_z3 ( z3::context &  ctx,
z3::expr_vector &  variable_names 
)
pure virtual

The shape of a polytope is defined through the set its boundaries. The representation of a polytope's boudaries is dependent on its shape and surfaces. This function takes the polytope dependent boundary representation and puts them into z3 format. A z3 representation is necessary for the solver to ensure that a solver only consideres coordinates within the boundaries of a polytope.

◆ depth

int polytope::depth
protected

This number represents the progress of the algorithm and is used to abort calculation at some point. The higher depth, the more steps have been taken to come to this point. What depth is actually representing is dependent (1) the concrete polytope used (2) the splitting heuristic.
