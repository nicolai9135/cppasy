cppasy

cppasy is a C++ parameter synthesis tool derived from PaSyPy. For a detailed description of this project, read our publication

Feature Overview

CMake Build

To build this project, execute the following in the root directory of this project

$ mkdir build
$ cd build
$ cmake ..
$ cmake --build . [--target <target>]

Usage

GUI

To use the GUI, simply execute $ ./gui. To understand available options, we recommend reading the publication.

CLI

To use the CLI, execute $ ./cli. Execute $ ./cli --help for information on possible arguments. To understand available options, we recommend reading the publication.

Documentation

The documentation of this project is hosted here. Note, that the GUI is missing a documentation as the code it is not expected to be used by anyone else than myself. If you want to substantially change the GUI, I recommend abandoning wxWidgets and using a different library.
